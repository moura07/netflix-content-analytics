\# Camada Bronze — Dados Brutos



Esta camada contém os dados brutos utilizados no projeto, exatamente como foram disponibilizados pela fonte original.



\## Fonte dos Dados



Dataset: Netflix Movies and TV Shows  

Fonte: Kaggle  

https://www.kaggle.com/datasets/shivamb/netflix-shows



\## Conteúdo



\- `netflix\_titles.csv`

\- Dados sem qualquer tipo de limpeza ou transformação



\## Versionamento



Os dados brutos são versionados neste projeto por se tratarem de arquivos leves e públicos, facilitando a reprodução do projeto.



\## Responsabilidade da Camada



\- Preservar os dados na forma original

\- Garantir rastreabilidade da fonte

\- Servir como ponto inicial do pipeline
